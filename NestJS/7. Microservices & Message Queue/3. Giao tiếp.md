Microservices trong NestJS có hai pattern giao tiếp chính, được đánh dấu bằng decorator trong `Controller`.

### 3.1. Request-Response (`@MessagePattern`)
Phù hợp khi client cần câu trả lời (tương tự HTTP). Client gửi request và đợi response.

```typescript
// math.controller.ts
import { Controller } from '@nestjs/common';
import { MessagePattern } from '@nestjs/microservices';

@Controller()
export class MathController {
  @MessagePattern({ cmd: 'sum' }) // Pattern định danh
  accumulate(data: number[]): number {
    return (data || []).reduce((a, b) => a + b);
  }
}
```
- Hỗ trợ **async** và trả về **Observable**.

### 3.2. Event-Based (`@EventPattern`)
Phù hợp khi bạn muốn gửi thông báo hoặc kích hoạt hành động nền mà không cần phản hồi (Fire-and-Forget).

```typescript
// user.controller.ts
import { Controller } from '@nestjs/common';
import { EventPattern } from '@nestjs/microservices';

@Controller()
export class UserController {
  @EventPattern('user_created')
  async handleUserCreated(data: Record<string, unknown>) {
    // Gửi email chào mừng, log, v.v.
    console.log('User created:', data);
  }
}
```
